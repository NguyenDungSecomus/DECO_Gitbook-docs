# What Is The Recommendation For Self-Uploading Templates?

When using DECO, you can select a pre-designed template from our gallery, or **upload your own design to use it as a label/badge/trust badge.**

If you would like to upload templates, here are **some recommendations** from DECO Designer team for you:

* File supported: PNG, JPG, JPEG, WEBP, SVG, GIF, etc. But we recommend you use: **SVG**, or **WEBP** to have the best image quality.
* Dimension: About **200x200px**
* File Size: **<60KB** (or compress it as small as possible to maintain the sitespeed)
