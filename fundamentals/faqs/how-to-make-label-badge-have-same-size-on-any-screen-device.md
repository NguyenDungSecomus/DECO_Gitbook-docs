# How To Make Label/Badge Have Same Size on Any Screen/Device ?

Open the label/badge you want to adjust size > On <mark style="color:blue;">**Design**</mark> tab, naviagte to <mark style="color:blue;">**Size**</mark> section > Here, you will see 2 options (as in the screenshot below) > let's select: <mark style="color:blue;">**Change size separately (pixel)**</mark>.&#x20;

{% hint style="info" %}
FYI, with the option: Change size separately (pixel), all of your size setups will be saved as Pixel (px), so when you want <mark style="color:blue;">**your label/badge to have the same size on any screen/device (even though whatever your product image's size)**</mark>, let's choose this option!
{% endhint %}
