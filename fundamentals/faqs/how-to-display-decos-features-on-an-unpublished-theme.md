# How To Display DECO's Features on An Unpublished Theme?

## 1. What Is An Unpublished Theme?&#x20;

* **An unpublished theme/A draft theme**: A theme that has been uploaded or created but is <mark style="color:blue;">**not currently active or visible to customers visiting the online store**</mark>. It exists within your theme library but is not applied to the storefront.

<figure><img src="../../.gitbook/assets/Screenshot_19 (1).png" alt=""><figcaption></figcaption></figure>

## 2. What Steps?

**Step 1**: Create DECO's Features: Label/ Badge/ Banner/ Trust Badge ( if you have, skip this step)&#x20;

**Step 2**: Navigate to _<mark style="color:blue;">**Sale Channels**</mark>_ >> Select: _<mark style="color:blue;">**Online Store**</mark>_

**Step 3**: Next, you will see _<mark style="color:blue;">**Themes**</mark>_ screen, select an unpublished theme in your _<mark style="color:blue;">**Theme library**</mark>_ >> Hit on _<mark style="color:blue;">**Customize**</mark>_ button of the selected theme.

<figure><img src="../../.gitbook/assets/image (501).png" alt=""><figcaption></figcaption></figure>

**Step 4**: Then, _<mark style="color:blue;">**choose App embeds tab**</mark>_ >> Search: DECO and _<mark style="color:blue;">**enable the toggle**</mark>_ of DECO >> Click the _<mark style="color:blue;">**Save**</mark>_ button.

<figure><img src="../../.gitbook/assets/image (502).png" alt=""><figcaption></figcaption></figure>

**Step 5**: To **check how DECO's features look on your store**, click on <mark style="color:blue;">**...**</mark> icon on the top-left, next to the theme name >> Select the _<mark style="color:blue;">**Preview**</mark>_ option

<figure><img src="../../.gitbook/assets/image (503).png" alt=""><figcaption></figcaption></figure>

{% embed url="https://youtu.be/GZpTrcmbIFg" %}

## 3. Why It's Not Showing Up?

There are 2 possible reasons:&#x20;

### 1. The DECO's snippet Has Not Been Enabled In Your Unpublished Theme

<figure><img src="../../.gitbook/assets/image (505).png" alt=""><figcaption></figcaption></figure>

⇒ To resolve it, please navigate to <mark style="color:blue;">**Sale Channels**</mark> > <mark style="color:blue;">**Online Store**</mark> > Click on: <mark style="color:blue;">**Customize**</mark> button > Select: <mark style="color:blue;">**App embeds**</mark> > Make sure that <mark style="color:blue;">**DECO's toggle has been Enabled**</mark> > Hit on: <mark style="color:blue;">**Save**</mark> button

### 2. Some technical issues

⇒ In case you have enabled the DECO's snippet, but your label/badge is not displaying, there might be a technical issue. Kindly no worries, <mark style="color:blue;">**chat with us via the live-chat box to resolve quickly!**</mark>
