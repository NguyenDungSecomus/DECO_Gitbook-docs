# 🪁 Transfer Settings

{% hint style="info" %}
When you "duplicate" your label/ badge, only the "Design" and "Display" settings are copied. So to copy the settings of "Products", let's try: "Transfer Settings"
{% endhint %}

## How "Transfer Settings" Work?

{% embed url="https://www.youtube.com/watch?v=9jXqTxJDDxI" %}

* By using this feature, you can easily copy the settings from a previous label or badge into the one you're currently editing.
