# 🌏 Show Badge Based On Store Languages

{% hint style="info" %}
**NOTE**: This feature is available for _<mark style="color:$primary;">**Badge only**</mark>_
{% endhint %}

## Where It Is?

* Navigate to _<mark style="color:$primary;">**"Labels & Badges"**</mark>_ on the navigation menu&#x20;
* Click the _<mark style="color:$primary;">**pencil icon**</mark>_ to open a badge

<figure><img src="../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>

* Next, move to "Display" tab >> Scroll down to: "_<mark style="color:$primary;">**Show badges on store languages**</mark>_" section

<figure><img src="../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

## Choose languages

* Select: _<mark style="color:$primary;">**"All Languages"**</mark>_: The badge will be _<mark style="color:$primary;">**visible in all languages**</mark>_ on your website.
* Select _<mark style="color:$primary;">**"Custom Languages"**</mark>_<mark style="color:$primary;">:</mark> The badge will _<mark style="color:$primary;">**only appear**</mark>_ when _<mark style="color:$primary;">**customers select specific languages you set up in the DECO app.**</mark>_

<figure><img src="../.gitbook/assets/image (45).png" alt=""><figcaption></figcaption></figure>
