# 🎁 DECO Trust Badges

**Trust Badge**: Symbols or Icons display on website to **assure visitors of security, reliability, and credibility**, often used to increase customer confidence and improve conversion rates.

Ex:&#x20;

<figure><img src="../.gitbook/assets/image (572).png" alt=""><figcaption></figcaption></figure>

## How To Create A DECO's Trust Badge?

## STEP 1: Select A Trust Badge Type

* Hit the button: "Create trust badge" :

<figure><img src="../.gitbook/assets/create new trust badeg.png" alt=""><figcaption></figcaption></figure>

* Then, select 1 trust badge type to get started:

<figure><img src="../.gitbook/assets/trust badge type.png" alt=""><figcaption></figcaption></figure>

## STEP 2: **Customize Content**&#x20;

* **"Content" tab:** Select the icon, Add the Icon Title, Add the Heading Text, Choose the Arrangement (Horizontal/Vertical), and Link Option

**NOTE**: Hit on "Upload image" to upload your design (File supported: Png, Svg, Gif, Webp)

P/s: If you need a customized icon, feel free to chat with us anytime!

<figure><img src="../.gitbook/assets/Screenshot_2 (1).png" alt=""><figcaption></figcaption></figure>

## STEP 3: Customize Design

* You can choose whether your trust badge icon **HAVING COLOR/ COLORLESS**
* Besides, you can custom **"Icon size",** "**Gap between icons**" & choose its "I**con align"**

{% hint style="info" %}
With these customizations, you can do it **DIFFERENTLY** on **DESKTOP** and **MOBILE** devices.
{% endhint %}

<figure><img src="../.gitbook/assets/image (315).png" alt=""><figcaption></figcaption></figure>

* Also, you can custom the **background color** and choose some **other settings** here:

<figure><img src="../.gitbook/assets/image (318).png" alt=""><figcaption></figcaption></figure>

## STEP 4: Select Placement

* **Devices:** Select whether to display the badge on desktop, mobile, or both.
* **Schedule:** Set a start and end date to control the display period.
*   **Pages:** Select the pages where you want the badge to appear.&#x20;

    ⇒ For product pages, you can use:

    \+ Default position (below the "Buy" button)&#x20;

    \+ Custom position (App block): See the detailed instructions below.

<figure><img src="../.gitbook/assets/Screenshot_3 (1).png" alt=""><figcaption></figcaption></figure>

### How to display DECO's Trust Badge Anywhere?

<figure><img src="../.gitbook/assets/Screenshot_4 (1).png" alt=""><figcaption></figcaption></figure>

Step 1: Select the position name for it, for example, DECO Trust Badge 1.

Step 2: Click "Save" button to save the name first

Step 3: Hit the "Add block to theme" button to go to your theme, and add a DECO Trust badge block to your store.

Step 4: Once done, please click on the block, select the Position name that you have set up, as example, DECO Trust Badge 1.

Step 5: Move it to anywhere you want -> Check the preview&#x20;

And don't forget to save your settings.

{% hint style="info" %}
**NOTE:** To see the preview, please ensure that <mark style="color:blue;">**the app embed has been turned on**</mark> & you <mark style="color:blue;">**choose the correct name that you assigned**</mark> for the trust badge icon set in-app
{% endhint %}
