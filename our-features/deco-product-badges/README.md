# 🚀 DECO Product Badges

## 🔖 **What is a DECO Product Badge?**

* A Badge is any graphic element laid **INSIDE/OUTSIDE OF AN IMAGE** to provide additional information, context, or identification for your products.

<figure><img src="../../.gitbook/assets/inside images.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/Outside images.png" alt=""><figcaption></figcaption></figure>

## 🛠️ Where to create a badge?

* There are 2 WAYS:&#x20;

1/ From the DECO's home page,&#x20;

* To _**create**_ a new badge: Click: <mark style="color:blue;">"</mark>_<mark style="color:blue;">**Create"**</mark>_ button in the Label & Badge block to get started.
* To _**quickly view**_ all badges created: Click: _<mark style="color:blue;">**"Manage"**</mark>_

<figure><img src="../../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>

2/ Or navigate to <mark style="color:$primary;">"</mark>_<mark style="color:$primary;">**Labels &**</mark>_ _<mark style="color:$primary;">**Badges**</mark>_<mark style="color:$primary;">"</mark> on the left navigation menu >> Then, click <mark style="color:$primary;">"</mark>_<mark style="color:$primary;">**Create badge**</mark>_<mark style="color:$primary;">"</mark> to get started

<figure><img src="../../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure>

## ⚙️Choose The Badge Type

* Then, select 1 type of the badge >> Click "_**Create**_" button to get started:

1/ <mark style="color:blue;">**Text badge**</mark><mark style="color:$info;">**:**</mark>  The _**editable type**_, which **you can edit anything**, like shape, background color, content, text color, etc

2/ <mark style="color:blue;">**Image badge**</mark>: The _**pre-made**_ type, which **you can't edit**. However, if you require any modifications to the template, please don't hesitate to contact our support team for assistance.&#x20;

<figure><img src="../../.gitbook/assets/image (609).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
<mark style="color:blue;">**NOTE**</mark>: **Once you selected the badge type (Image/Text)** in this screen, **you couldn't change the badge type in the next step** (if you would like to change, you need to create a new badge)
{% endhint %}
