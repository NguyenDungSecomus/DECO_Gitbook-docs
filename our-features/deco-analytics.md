# 📈 DECO Analytics

{% hint style="info" %}
Data will start showing up in the Analytics report from the moment you upgraded + having at least 1 active badge displaying on your website already!
{% endhint %}

## 🎯OVERVIEWS

Track key e-commerce metrics for your active products, which are using DECO badges:

* **Add-to-cart** totals
* **Order** totals
* **Revenue** totals

<figure><img src="../.gitbook/assets/test (1).png" alt=""><figcaption><p>Example Of The DECO Report</p></figcaption></figure>

## ⚙️ PREPARATIONS

Before accessing the DECO Analytics Report, ensure you have:

✅ **DECO app embed**: Active

✅ **DECO badges**: At least 1&#x20;

✅ **DECO plan**: From Starter

## &#x20;💡HOW TO USE

* Use the filter to choose the appropriate view:

<figure><img src="../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure>



* In case you need to **RE-CHECK** data in a specific time frame: Click the "_<mark style="color:blue;">**calendar icon**</mark>_" >> Choose the timeframe >> Click "<mark style="color:blue;">**Apply**</mark>" button

<figure><img src="../.gitbook/assets/image (562).png" alt=""><figcaption></figcaption></figure>

{% hint style="warning" %}
For daily tracking, **no date range selection is needed**—the report shows current data by default.
{% endhint %}

## ✏️ NOTICES

| Data processing can take 24-48h. During that time, data in your reports may change.                                          |
| ---------------------------------------------------------------------------------------------------------------------------- |
| Data refreshes **daily** at 9 AM and 12 PM (GMT+7), including weekends                                                       |
| Data **ONLY** is collected **from the date you upgraded** (the previous time won't be counted)                               |
| During the time you **DOWNGRADED** to DECO FREE plan, the data will be **paused** to collect, **until** you **charge again** |
